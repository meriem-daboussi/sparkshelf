# ⚙️ SparkShelf

**SparkShelf** is an interactive idea discovery tool built for engineering and robotics enthusiasts.  
Inspired by swipe-style apps, it helps teams and students quickly explore, like, and save project ideas that fit their goals or constraints.

# Features
- Swipe left/right to explore and match with robotics ideas  
- Save and manage liked ideas locally (no login required)  
- Toggle **dark/light** mode with persistent theme memory  
- Expandable catalog — easy to add new projects or tools  
- Fully static: deploy anywhere (GitHub Pages, Netlify, etc.)

# Built With
HTML • CSS • JavaScript (Vanilla, modular structure)
